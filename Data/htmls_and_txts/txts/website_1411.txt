

Цифровая кафедра | Программа дополнительной профессиональной переподготовки «Информационные технологии бизнес-анализа»   
  





  ПоступающимСтудентамВыпускникамМедиатекаУниверситетОбразованиеНаукаБизнесуУниверситетПрограмма «Приоритет 2030»×2Версия для слабовидящихЛичный кабинетПрограмма дополнительной профессиональной подготовки «Информационные технологии бизнес-анализа»НИТУ МИСИС приглашает студентов пройти бесплатное обучение по программам профессиональной подготовки с присвоением квалификации в рамках проекта «Цифровые кафедры».


На рынке труда существует острый дефицит специалистов, работающих с различными информационными системами, предназначенными для управления, анализа и принятия решений в бизнес-сфере и управлении. 
В основе их работы лежат комплексные решения для автоматизации расчета и наглядной демонстрации ключевых показателей системы, представляющие собой комплекс процедур, методов и средств, охватывающих основные задачи и функции управления бизнес-деятельностью и обеспечивающие принятие решений на основе информации, получаемой с помощью информационных технологий.
Целью подготовки слушателей программы «Информационные технологии бизнес-анализа» является получение компетенций, необходимых для выполнения нового вида профессиональной деятельности и приобретение новой квалификации «Системный бизнес аналитик».
Полученные универсальные компетенции расширят спектр возможностей для выпускников, обучавшихся по специальностям и направлениям подготовки не отнесенным к IT-сфере, по выбору профессии.

Запись на программу до 21.08.2023.
Подать заявку

Преимущества программы


получение навыков экономического анализа на основе информационных систем для не специалистов в области ИТ;
возможность ознакомиться с назначением, особенностями
настройками и функциональными возможностями основных модулей одной из современных BI-систем;
освоение методов и технологий использования современных информационноаналитических систем в управлении бизнес-системами.

Высокий уровень преподавателей участвующих в учебном процессе и общение с экспертами из бизнеса по профильным направлениям;

диплом о профессиональной переподготовке с присвоением новой квалификации — «Системный бизнес аналитик».


К обучению приглашаются


бакалавры не ранее 2 курса;
специалисты не ранее 3 курса;
магистры 1, 2 курса, обучающиеся по направлениям не ИТ-профиля.


Получаемая квалификация и результат обучения

Диплом о профессиональной переподготовке с присвоением новой квалификации «Системный бизнес аналитик».
Примечание: выдача диплома возможна только для обучающихся, имеющих диплом о высшем образовании. Для студентов бакалавриата и специалитета (не выпускного курса) выдача диплома будет произведена только при завершении основной профессиональной образовательной программы.

Сроки обучения

Срок освоения программы составляет 260 академических часов.

начало обучения в октябре 2023 года;
окончание — в октябре 2024 года.


Формат обучения и содержание программы

Учебный процесс организуется в очном формате, с применением электронного обучения, с помощью дистанционных образовательных технологий.
Реализация программы обеспечивается научно-педагогическими кадрами университета, высококвалифицированными специалистами ИТ-сферы и других областей, касающихся профессиональных компетенций, приобретаемых в процессе освоения программы.
Программа состоит из 5 образовательных модулей:
Модуль 1. Основы современного бизнес-анализа. 

Применение информационных систем для анализа данных.

Модуль 2. Комплекс инструментов для бизнес-анализа. 

Power Query и Power Pivot

Модуль 3. Анализ данных при помощи Microsoft Power BI
Модуль 4. Современные хранилища данных, аналитика больших данных и машинное обучение на SQL
Модуль 5. Технологии формирования многомерной и интерактивной отчетности
Итог: защита группового проекта.

Формируемые компетенции

В ходе освоения программы слушатели приобретают следующие профессиональные компетенции:

Дорабатывает конфигурации и модули ИС (информационные системы) предприятий.
Использует большие данные при реализации проектов в области управления финансовой информацией.
Использует программные и технические средства для визуализации больших данных.
